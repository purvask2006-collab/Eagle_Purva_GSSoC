# Contributing to Eagle

Thank you for your interest in contributing to Eagle!

## My First Contribution (Zero to PR in 15 Minutes)

Never contributed to open source before? Follow these steps:

**Step 1: Fork the repository**

Click the Fork button at the top right of this page.

**Step 2: Clone your fork**

```bash
git clone https://github.com/YOUR_USERNAME/Eagle-Real-Time-Semantic-Surveillance-using-Detection-Tracking-Reasoning.git
cd Eagle-Real-Time-Semantic-Surveillance-using-Detection-Tracking-Reasoning
```

**Step 3: Create a branch**

```bash
git checkout -b your-branch-name
```

**Step 4: Make your changes**

Edit the files you want to improve.

**Step 5: Commit your changes**

```bash
git status
git add <your-changed-files>
git commit -m "describe what you changed"
```

**Step 6: Push and open a PR**

```bash
git push origin your-branch-name
```

Then go to the original repo and click **Compare and pull request**.
